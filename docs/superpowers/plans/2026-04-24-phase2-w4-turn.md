# Phase 2 W4: TURN Relay — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** TURN (RFC 5766) relay 経由での通信を追加。両端が対称 NAT で probe 失敗する場合でも TURN サーバ経由で接続成立させる。

**Architecture:** `prdt-nat-traversal::turn::TurnClient` が Allocate/CreatePermission/Send Indication/Data Indication を実装。`TurnRelaySocket` が `UdpSocket` 互換 API(send_to/recv_from)で Send/Data Indication を透過 wrap/unwrap。`CustomUdpTransport::bind_with_relay` が relay mode コンストラクタ。signaling-client が turn_url 指定時に Allocate → Relay candidate を signaling に送る。

**Tech Stack:** Rust (tokio 1.40, stun_codec 0.3 `rfc5766` module, bytecodec 0.4, hmac 0.12, sha1 0.10, md5 0.7, 既存 prdt-* crates)

**Spec:** `docs/superpowers/specs/2026-04-24-phase2-w4-turn-design.md`

**重要:** `stun_codec::rfc5766` に TURN methods + attributes + pre-composed `Attribute` enum が全て揃っているので、プロトコル wire 層は最小コード。自前実装は auth MESSAGE-INTEGRITY 計算とクライアント state 管理のみ。

```rust
//! TURN (RFC 5766) client.
//!
//! Scope: Allocate (with long-term credential auth), CreatePermission,
//! Send Indication encode, Data Indication decode. Refresh and ChannelBind
//! are out of scope (Phase 2 W5+).

use std::net::SocketAddr;
use std::time::Duration;
use url::Url;

#[derive(Debug, Clone)]
pub struct TurnConfig {
    pub server_addr: SocketAddr,
    pub username: String,
    pub password: String,
    pub requested_lifetime: Duration,
}

impl TurnConfig {
    /// Parse a `turn://user:pass@host:port` URL into a config.
    /// DNS lookup for the host is performed.
    pub async fn from_url(url: &Url) -> Result<Self, TurnError> {
        if url.scheme() != "turn" {
            return Err(TurnError::BadUrl(format!(
                "unsupported scheme: {}",
                url.scheme()
            )));
        }
        let username = url.username().to_string();
        let password = url.password().unwrap_or("").to_string();
        if username.is_empty() || password.is_empty() {
            return Err(TurnError::BadUrl("username and password required".into()));
        }
        let host = url
            .host_str()
            .ok_or_else(|| TurnError::BadUrl("missing host".into()))?;
        let port = url.port().unwrap_or(3478);
        let server_addr = tokio::net::lookup_host(format!("{host}:{port}"))
            .await
            .map_err(|e| TurnError::BadUrl(format!("resolve: {e}")))?
            .next()
            .ok_or_else(|| TurnError::BadUrl("no addrs".into()))?;
        Ok(Self {
            server_addr,
            username,
            password,
            requested_lifetime: Duration::from_secs(600),
        })
    }
}

#[derive(thiserror::Error, Debug)]
pub enum TurnError {
    #[error("io: {0}")]
    Io(#[from] std::io::Error),
    #[error("bad turn URL: {0}")]
    BadUrl(String),
    #[error("timeout during {stage}")]
    Timeout { stage: &'static str },
    #[error("server error: code={code} reason={reason}")]
    Server { code: u16, reason: String },
    #[error("protocol: {0}")]
    Protocol(String),
    #[error("bad auth: {0}")]
    Auth(String),
}
```

```rust
use bytecodec::{DecodeExt, EncodeExt};
use prdt_nat_traversal::turn::{TurnClient, TurnConfig, TurnError};
use std::net::SocketAddr;
use std::time::Duration;
use stun_codec::rfc5389::errors::Unauthorized;
use stun_codec::rfc5389::attributes::{ErrorCode, MessageIntegrity, Nonce, Realm, Username};
use stun_codec::rfc5766::attributes::{Lifetime, RequestedTransport, XorRelayedAddress};
use stun_codec::rfc5766::methods::ALLOCATE;
use stun_codec::{
    Message, MessageClass, MessageDecoder, MessageEncoder,
};
use tokio::net::UdpSocket;

/// Minimal mock TURN server: answers Allocate with 401 on first attempt, then
/// 200 Success on retry with MESSAGE-INTEGRITY. Returns its bound addr +
/// fixed "relayed addr" it will hand out.
async fn spawn_mock_turn(
    username: &'static str,
    password: &'static str,
) -> (SocketAddr, SocketAddr) {
    let realm = "prdt-test";
    let nonce = "01234567";
    let relayed: SocketAddr = "127.0.0.1:55555".parse().unwrap();

    let socket = UdpSocket::bind("127.0.0.1:0").await.unwrap();
    let addr = socket.local_addr().unwrap();
    tokio::spawn(async move {
        let mut buf = [0u8; 1500];
        loop {
            let Ok((n, src)) = socket.recv_from(&mut buf).await else { break };
            // decode as stun_codec::rfc5766::Attribute
            let mut decoder = MessageDecoder::<stun_codec::rfc5766::Attribute>::new();
            let Ok(Ok(req)) = decoder.decode_from_bytes(&buf[..n]) else { continue };
            if req.method() != ALLOCATE || req.class() != MessageClass::Request { continue; }
            let has_mi = req.get_attribute::<MessageIntegrity>().is_some();
            if !has_mi {
                // Send 401 challenge
                let mut resp = Message::<stun_codec::rfc5766::Attribute>::new(
                    MessageClass::ErrorResponse,
                    ALLOCATE,
                    req.transaction_id(),
                );
                resp.add_attribute(stun_codec::rfc5766::Attribute::from(
                    ErrorCode::new(401, "Unauthorized".to_string()).unwrap(),
                ));
                resp.add_attribute(stun_codec::rfc5766::Attribute::from(
                    Realm::new(realm.to_string()).unwrap(),
                ));
                resp.add_attribute(stun_codec::rfc5766::Attribute::from(
                    Nonce::new(nonce.to_string()).unwrap(),
                ));
                let mut enc = MessageEncoder::<stun_codec::rfc5766::Attribute>::new();
                let bytes = enc.encode_into_bytes(resp).unwrap();
                let _ = socket.send_to(&bytes, src).await;
            } else {
                // Validate MI: key = MD5(user:realm:password)
                use md5::{Md5, Digest};
                let key_input = format!("{}:{}:{}", username, realm, password);
                let mut h = Md5::new();
                h.update(key_input.as_bytes());
                let key = h.finalize();
                // stun_codec MessageIntegrity::verify is awkward; in mock we just trust
                // the presence of MI + matching username, since the client test
                // is the source of truth for crypto.
                if req.get_attribute::<Username>().map(|u| u.name()) != Some(username) {
                    continue;
                }
                let _ = &key; // silence unused warn
                // Send AllocateSuccess with XOR-RELAYED-ADDRESS + LIFETIME
                let mut resp = Message::<stun_codec::rfc5766::Attribute>::new(
                    MessageClass::SuccessResponse,
                    ALLOCATE,
                    req.transaction_id(),
                );
                resp.add_attribute(stun_codec::rfc5766::Attribute::from(
                    XorRelayedAddress::new(relayed),
                ));
                resp.add_attribute(stun_codec::rfc5766::Attribute::from(
                    Lifetime::new(Duration::from_secs(600)).unwrap(),
                ));
                let mut enc = MessageEncoder::<stun_codec::rfc5766::Attribute>::new();
                let bytes = enc.encode_into_bytes(resp).unwrap();
                let _ = socket.send_to(&bytes, src).await;
            }
        }
    });
    (addr, relayed)
}
```

```rust
use bytecodec::{DecodeExt, EncodeExt};
use rand_core::{OsRng, RngCore};
use std::sync::Arc;
use stun_codec::rfc5389::attributes::{
    ErrorCode, MessageIntegrity, Nonce, Realm, Username,
};
use stun_codec::rfc5766::attributes::{
    Lifetime, RequestedTransport, XorRelayedAddress,
};
use stun_codec::rfc5766::methods::ALLOCATE;
use stun_codec::{
    Message, MessageClass, MessageDecoder, MessageEncoder, TransactionId,
};
use tokio::net::UdpSocket;

pub struct TurnClient {
    socket: Arc<UdpSocket>,
    config: TurnConfig,
    realm: Option<String>,
    nonce: Option<String>,
    relayed_addr: Option<SocketAddr>,
}

impl TurnClient {
    pub fn new(socket: Arc<UdpSocket>, config: TurnConfig) -> Self {
        Self { socket, config, realm: None, nonce: None, relayed_addr: None }
    }

    pub fn realm(&self) -> Option<&str> { self.realm.as_deref() }
    pub fn nonce(&self) -> Option<&str> { self.nonce.as_deref() }
    pub fn relayed_addr(&self) -> Option<SocketAddr> { self.relayed_addr }

    /// Send Allocate request, handle 401 challenge, retry with MESSAGE-INTEGRITY.
    /// Stores relayed_addr, realm, nonce on success.
    pub async fn allocate(&mut self, timeout: Duration) -> Result<SocketAddr, TurnError> {
        // Attempt 1: no auth
        let txn = random_transaction_id();
        let mut req1 = Message::<stun_codec::rfc5766::Attribute>::new(
            MessageClass::Request,
            ALLOCATE,
            txn,
        );
        req1.add_attribute(stun_codec::rfc5766::Attribute::from(
            RequestedTransport::new(17), // UDP
        ));
        req1.add_attribute(stun_codec::rfc5766::Attribute::from(
            Lifetime::new(self.config.requested_lifetime).unwrap(),
        ));
        let bytes = encode(&req1)?;
        self.socket.send_to(&bytes, self.config.server_addr).await?;

        let resp = recv_with_timeout(&self.socket, timeout).await?;
        // Decode
        let resp_msg: Message<stun_codec::rfc5766::Attribute> = decode(&resp)?;
        if resp_msg.class() == MessageClass::ErrorResponse {
            let code = resp_msg.get_attribute::<ErrorCode>()
                .map(|e| e.code()).unwrap_or(0);
            if code == 401 {
                let realm = resp_msg.get_attribute::<Realm>()
                    .ok_or_else(|| TurnError::Protocol("401 without REALM".into()))?
                    .text().to_string();
                let nonce = resp_msg.get_attribute::<Nonce>()
                    .ok_or_else(|| TurnError::Protocol("401 without NONCE".into()))?
                    .value().to_string();
                self.realm = Some(realm.clone());
                self.nonce = Some(nonce.clone());
                // Attempt 2: with USERNAME + REALM + NONCE + MESSAGE-INTEGRITY
                return self.allocate_retry_with_auth(timeout).await;
            } else {
                return Err(TurnError::Server {
                    code,
                    reason: format!("{:?}", resp_msg.get_attribute::<ErrorCode>()),
                });
            }
        }
        // Unexpected: success without challenge? Some servers allow unauth.
        if let Some(relayed) = resp_msg.get_attribute::<XorRelayedAddress>() {
            self.relayed_addr = Some(relayed.address());
            return Ok(relayed.address());
        }
        Err(TurnError::Protocol("no XOR-RELAYED-ADDRESS in success".into()))
    }

    async fn allocate_retry_with_auth(&mut self, timeout: Duration) -> Result<SocketAddr, TurnError> {
        let realm = self.realm.clone().ok_or_else(|| TurnError::Auth("no realm".into()))?;
        let nonce = self.nonce.clone().ok_or_else(|| TurnError::Auth("no nonce".into()))?;
        let txn = random_transaction_id();
        let mut req = Message::<stun_codec::rfc5766::Attribute>::new(
            MessageClass::Request,
            ALLOCATE,
            txn,
        );
        req.add_attribute(stun_codec::rfc5766::Attribute::from(
            RequestedTransport::new(17),
        ));
        req.add_attribute(stun_codec::rfc5766::Attribute::from(
            Lifetime::new(self.config.requested_lifetime).unwrap(),
        ));
        req.add_attribute(stun_codec::rfc5766::Attribute::from(
            Username::new(self.config.username.clone())
                .map_err(|e| TurnError::Auth(format!("bad username: {e}")))?,
        ));
        req.add_attribute(stun_codec::rfc5766::Attribute::from(
            Realm::new(realm.clone())
                .map_err(|e| TurnError::Auth(format!("bad realm: {e}")))?,
        ));
        req.add_attribute(stun_codec::rfc5766::Attribute::from(
            Nonce::new(nonce.clone())
                .map_err(|e| TurnError::Auth(format!("bad nonce: {e}")))?,
        ));

        // MESSAGE-INTEGRITY computed as HMAC-SHA1 over the message bytes up to
        // and including the MI attribute header, with key = MD5(user:realm:password).
        // stun_codec's MessageIntegrity::new_long_term_credential handles this.
        let mi = MessageIntegrity::new_long_term_credential(
            &req,
            &self.config.username,
            &realm,
            &self.config.password,
        ).map_err(|e| TurnError::Auth(format!("MI: {e:?}")))?;
        req.add_attribute(stun_codec::rfc5766::Attribute::from(mi));

        let bytes = encode(&req)?;
        self.socket.send_to(&bytes, self.config.server_addr).await?;

        let resp_bytes = recv_with_timeout(&self.socket, timeout).await?;
        let resp_msg: Message<stun_codec::rfc5766::Attribute> = decode(&resp_bytes)?;
        if resp_msg.class() != MessageClass::SuccessResponse {
            let code = resp_msg.get_attribute::<ErrorCode>()
                .map(|e| e.code()).unwrap_or(0);
            return Err(TurnError::Server {
                code,
                reason: "allocate retry failed".into(),
            });
        }
        let relayed = resp_msg.get_attribute::<XorRelayedAddress>()
            .ok_or_else(|| TurnError::Protocol("success without XOR-RELAYED".into()))?
            .address();
        self.relayed_addr = Some(relayed);
        Ok(relayed)
    }
}

fn random_transaction_id() -> TransactionId {
    let mut id = [0u8; 12];
    OsRng.fill_bytes(&mut id);
    TransactionId::new(id)
}

fn encode(msg: &Message<stun_codec::rfc5766::Attribute>) -> Result<Vec<u8>, TurnError> {
    let mut e = MessageEncoder::<stun_codec::rfc5766::Attribute>::new();
    e.encode_into_bytes(msg.clone())
        .map_err(|err| TurnError::Protocol(format!("encode: {err}")))
}

fn decode(bytes: &[u8]) -> Result<Message<stun_codec::rfc5766::Attribute>, TurnError> {
    let mut d = MessageDecoder::<stun_codec::rfc5766::Attribute>::new();
    match d.decode_from_bytes(bytes) {
        Ok(Ok(m)) => Ok(m),
        Ok(Err(e)) => Err(TurnError::Protocol(format!("decode body: {e:?}"))),
        Err(e) => Err(TurnError::Protocol(format!("decode: {e}"))),
    }
}

async fn recv_with_timeout(
    socket: &UdpSocket,
    dur: Duration,
) -> Result<Vec<u8>, TurnError> {
    let mut buf = vec![0u8; 1500];
    match tokio::time::timeout(dur, socket.recv_from(&mut buf)).await {
        Ok(Ok((n, _))) => {
            buf.truncate(n);
            Ok(buf)
        }
        Ok(Err(e)) => Err(TurnError::Io(e)),
        Err(_) => Err(TurnError::Timeout { stage: "turn_recv" }),
    }
}
```

```rust
use stun_codec::rfc5766::attributes::{Data, XorPeerAddress};
use stun_codec::rfc5766::methods::{DATA, SEND};

/// One decoded Data Indication: opaque payload + the peer addr it came from.
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct DataIndication {
    pub peer: SocketAddr,
    pub data: Vec<u8>,
}

impl TurnClient {
    /// Wrap `data` in a Send Indication addressed to `peer` and send it to the
    /// TURN server. The TURN server will forward the inner `data` bytes to
    /// `peer` as raw UDP (provided CreatePermission was called for `peer`).
    /// Send Indications are NOT authenticated per RFC 5766 §10.
    pub async fn send_indication(
        &self,
        peer: SocketAddr,
        data: &[u8],
    ) -> Result<(), TurnError> {
        let txn = random_transaction_id();
        let mut msg = Message::<stun_codec::rfc5766::Attribute>::new(
            MessageClass::Indication,
            SEND,
            txn,
        );
        msg.add_attribute(stun_codec::rfc5766::Attribute::from(
            XorPeerAddress::new(peer),
        ));
        msg.add_attribute(stun_codec::rfc5766::Attribute::from(
            Data::new(data.to_vec())
                .map_err(|e| TurnError::Protocol(format!("data too large: {e:?}")))?,
        ));
        let bytes = encode(&msg)?;
        self.socket.send_to(&bytes, self.config.server_addr).await?;
        Ok(())
    }
}

/// If `bytes` is a TURN Data Indication, return Some(DataIndication). If it
/// isn't a valid TURN message (or is any other type), return Ok(None) so the
/// caller can fall back to raw UDP handling.
pub fn try_decode_data_indication(bytes: &[u8]) -> Result<Option<DataIndication>, TurnError> {
    let mut d = MessageDecoder::<stun_codec::rfc5766::Attribute>::new();
    let msg = match d.decode_from_bytes(bytes) {
        Ok(Ok(m)) => m,
        _ => return Ok(None), // not a valid STUN message
    };
    if msg.class() != MessageClass::Indication || msg.method() != DATA {
        return Ok(None);
    }
    let peer = msg.get_attribute::<XorPeerAddress>()
        .ok_or_else(|| TurnError::Protocol("DataIndication without XOR-PEER-ADDRESS".into()))?
        .address();
    let data = msg.get_attribute::<Data>()
        .ok_or_else(|| TurnError::Protocol("DataIndication without DATA".into()))?
        .data().to_vec();
    Ok(Some(DataIndication { peer, data }))
}
```

```rust
//! UdpSocket-compatible wrapper for TURN relay traffic. `send_to` wraps data in
//! a TURN Send Indication; `recv_from` auto-unwraps Data Indications and
//! returns `(bytes, real_peer_addr)`.

use crate::turn::{try_decode_data_indication, TurnClient, TurnConfig, TurnError};
use std::collections::HashSet;
use std::net::SocketAddr;
use std::sync::Arc;
use std::time::Duration;
use tokio::net::UdpSocket;
use tokio::sync::Mutex;

pub struct TurnRelaySocket {
    socket: Arc<UdpSocket>,
    client: Mutex<TurnClient>,
    permissions: Mutex<HashSet<SocketAddr>>,
    relayed: SocketAddr,
}

impl TurnRelaySocket {
    /// Bind a new UDP socket on `bind_addr` and allocate on the TURN server.
    pub async fn allocate(config: TurnConfig) -> Result<Self, TurnError> {
        let socket = Arc::new(UdpSocket::bind("0.0.0.0:0").await?);
        Self::allocate_with_socket(socket, config).await
    }

    /// Use an existing bound socket.
    pub async fn allocate_with_socket(
        socket: Arc<UdpSocket>,
        config: TurnConfig,
    ) -> Result<Self, TurnError> {
        let mut client = TurnClient::new(socket.clone(), config);
        let relayed = client.allocate(Duration::from_secs(5)).await?;
        Ok(Self {
            socket,
            client: Mutex::new(client),
            permissions: Mutex::new(HashSet::new()),
            relayed,
        })
    }

    pub fn local_addr(&self) -> std::io::Result<SocketAddr> {
        self.socket.local_addr()
    }

    pub fn relayed_addr(&self) -> SocketAddr {
        self.relayed
    }

    /// Add permission for `peer` if not already set.
    pub async fn ensure_permission(&self, peer: SocketAddr) -> Result<(), TurnError> {
        {
            let mut perms = self.permissions.lock().await;
            if perms.contains(&peer) {
                return Ok(());
            }
            perms.insert(peer);
        }
        let mut client = self.client.lock().await;
        client.create_permission(peer, Duration::from_secs(3)).await
    }

    /// Wrap `data` in a Send Indication and send via TURN. Caller is responsible
    /// for having called `ensure_permission(peer)`.
    pub async fn send_to(&self, data: &[u8], peer: SocketAddr) -> Result<usize, TurnError> {
        let client = self.client.lock().await;
        client.send_indication(peer, data).await?;
        Ok(data.len())
    }

    /// Read one datagram. If it's a TURN Data Indication, unwrap to
    /// `(inner_bytes, peer_addr)`. If not, return the raw source addr + all bytes.
    pub async fn recv_from(&self, buf: &mut [u8]) -> Result<(&[u8], SocketAddr), TurnError> {
        // We need to copy into `buf` then interpret. Return a slice of `buf`.
        // For simplicity (slice lifetime tied to buf), we write results in-place
        // when unwrapping a Data Indication.
        loop {
            let mut tmp = vec![0u8; buf.len().max(1500)];
            let (n, src) = self.socket.recv_from(&mut tmp).await?;
            if src == self.client.lock().await.config_server_addr() {
                // Could be a Data Indication — try to decode.
                if let Ok(Some(ind)) = try_decode_data_indication(&tmp[..n]) {
                    if ind.data.len() > buf.len() {
                        return Err(TurnError::Protocol(format!(
                            "data too big for buf: {} > {}",
                            ind.data.len(), buf.len()
                        )));
                    }
                    let len = ind.data.len();
                    buf[..len].copy_from_slice(&ind.data);
                    return Ok((&buf[..len], ind.peer));
                }
                // Not a Data Indication — could be a TURN control response.
                // Skip and keep reading.
                continue;
            }
            // Direct traffic from some other addr (rare in relay mode) — return raw.
            if n > buf.len() {
                return Err(TurnError::Protocol(format!(
                    "direct datagram too big for buf: {} > {}",
                    n, buf.len()
                )));
            }
            buf[..n].copy_from_slice(&tmp[..n]);
            return Ok((&buf[..n], src));
        }
    }
}
```
